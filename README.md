## Basic funcitons

- Register ship
- Record ship state (In or out)
- On cruise ships types
  - Get person
  - Get leaving people
  - Get incoming people
- On carrier ships
  - Register container
  - Container type, weight registration
  - Site picking for container
  - Container status (In tranit, In registration phase, On site)
- Site management
  - Get free space
  - Register container
  - Move container
  - Sign off container
